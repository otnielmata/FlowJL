# Flow JL Web

Front-end web do Flow JL construído com Next.js, React, TypeScript e Tailwind CSS para simular a operação completa de uma agência de lançamentos digitais.

## O que está pronto

- Layout moderno com sidebar recolhível, topbar, busca global e painel de notificações
- Suporte a tema claro, escuro e automático
- Perfis simulados com controle de permissões por módulo
- Estados globais de loading, erro, vazio e acesso negado
- Módulos entregues:
  - Dashboard
  - Estratégia Digital
  - Lançamentos
  - Cronogramas
  - Produção de Conteúdo
  - Social Media
  - Tráfego Pago
  - Operações
  - Aprovações
  - Inteligência Artificial
  - Relatórios
  - Configurações
- Tabelas com busca e filtros
- Gráficos com Recharts
- Kanban com drag and drop
- Calendário responsivo
- Formulários validados com React Hook Form + Zod
- Dados mockados prontos para futura integração com a API

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- TanStack Query
- Zustand
- Recharts
- DnD Kit
- Lucide React
- Sonner

## Como rodar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Validação executada

```bash
npm run lint
npm run build
```
